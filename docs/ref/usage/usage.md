# Usage

This section provides instructions on how to use the Wazuh Installation Assistant tools. It includes detailed guides and examples for each tool:

- **Certs Tool** ([certs-tool/certs-tool-usage.md](certs-tool/certs-tool-usage.md)): A guide on how to use the Certs Tool to manage certificates used by Wazuh.
- **Passwords Tool** ([passwords-tool/passwords-tool-usage.md](passwords-tool/passwords-tool-usage.md)): A guide on how to use the Passwords Tool to manage passwords related to Wazuh.
- **Installation Assistant** ([installation-assistant/ia-usage.md](installation-assistant/ia-usage.md)): A guide on how to use the Installation Assistant tool to install and configure Wazuh on Linux systems.
