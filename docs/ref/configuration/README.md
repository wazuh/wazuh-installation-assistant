# Configuration

This section covers the configuration options available for the Wazuh installation tools.

From version 5.x onwards, all scripts include the version and revision number in their filenames (e.g. `wazuh-install-5.9.9-1.sh`, `wazuh-certs-tool-5.9.9-1.sh`, `wazuh-passwords-tool-5.9.9-1.sh`).

- [Configuration Files](configuration-files.md) — Description of the configuration files used by the installation assistant and the certs tool.
- [Command Line Options](command-line-options.md) — Full reference of all command line options available for the installation assistant, certs tool, and passwords tool.
- [Environment Variables](environment-variables.md) — Information about environment variable support.
