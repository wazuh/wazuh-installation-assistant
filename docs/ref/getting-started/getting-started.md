# Introduction

This section introduces the Wazuh Installation Assistant tools.

- **Certs Tool** ([certs-tool-introduction.md](certs-tool/certs-tool-introduction.md)): A tool for managing certificates used by Wazuh.
- **Passwords Tool** ([passwords-tool-usage.md](passwords-tool/passwords-tool-usage.md)): A tool for managing passwords related to Wazuh.
- **Installation Assistant** ([installation-assistant-introduction.md](installation-assistant/ia-introduction.md)): A tool that facilitates the installation and configuration of Wazuh on Linux systems.
