# Certs Tool installation

The Certs Tool can be installed by downloading the `wazuh-certs-tool-5.0.0-1.sh` script from the Wazuh packages repository. You can do this by running the following command:

```bash
curl -sO https://packages.wazuh.com/5.0/wazuh-certs-tool-5.0.0-1.sh
```

For the certs-tool, the `config-5.0.0-1.yml` configuration file is also necessary, which can be downloaded with the following command:

```bash
curl -sO https://packages.wazuh.com/5.0/config-5.0.0-1.yml
```
